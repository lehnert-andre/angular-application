# angular-application

## Storybook

`$ npm i @storybook/cli -g`

`$ cd <angular project root>`

`$ getstorybook init`

`$ ng g application storybook --skipTests=true --style=scss`

Resources: 

- https://storybook.js.org/docs/guides/guide-angular/
- https://alligator.io/angular/storybook-angular/
